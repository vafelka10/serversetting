# 🛡️ Server Setup Script

Автоматическая настройка VPN-ноды на чистом Debian за одну команду.

---

## ⚡ Быстрый старт

```bash
bash <(curl -Ls https://raw.githubusercontent.com/vafelka10/serversetting/main/setup.sh)
```

---

## 📋 Что делает скрипт

| Шаг | Действие |
|-----|----------|
| 1 | Обновление системы (`apt update && upgrade`) |
| 2 | Установка `curl`, `ufw` |
| 3 | Создание файла подкачки 3 ГБ |
| 4 | Смена SSH порта |
| 5 | Настройка firewall (UFW) |
| 6 | Установка панели [3x-ui](https://github.com/mhsanaei/3x-ui) |
| 7 | Открытие порта панели в UFW |

---

## 🖥️ Требования

- Debian 12 / 13
- Чистая установка
- Доступ по SSH под root

---

## 📝 В процессе установки скрипт спросит

- **SSH порт** — новый порт вместо стандартного 22
- **Порт панели** — порт который ты указал при установке 3x-ui

---

## ⚠️ Важно

После смены SSH порта **не закрывай текущую сессию** — сначала проверь подключение в новом окне:

```bash
ssh root@твой_ip -p НОВЫЙ_ПОРТ
```
